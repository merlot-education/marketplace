Sie können Ihre Organisation alternativ zur unten bereitgestellten Maske auch über das Hochladen einer Selbstbeschreibung berarbeiten. Die Organisations-Selbstbeschreibung muss dafür signiert sein und als JSON-Datei vorliegen.
Als Föderationsadministrator können Sie außerdem Selbstbeschreibungen neuer Organisationen hochladen, um diese in MERLOT aufzunehmen.
